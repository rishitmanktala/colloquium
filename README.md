# 🏛️ Colloquium 2.0

**Official website for Colloquium 2.0** — the Model United Nations conference organized by Suncity School, Sector 37D, Gurugram.

📅 **22–23 August 2026** · 🌐 [projectcolloquium.in](https://projectcolloquium.in)

---

## ✨ Features

- 🎨 **Cinematic Dark Theme** — Immersive design with gold accents, custom fonts (Cinzel, Bebas Neue, Inter), and scroll-driven animations
- 📋 **Committee Showcase** — UNSC, UNFCCC, AIPPM, GFMC, IMI, CBI, Mock Trial, IPL Auction, International Press, and Founders' Circle
- 🎫 **Registration & Fees** — Delegate registration info and fee structure
- ❓ **FAQ Section** — Common delegate questions answered
- 📱 **Fully Responsive** — Mobile-first sidebar navigation with smooth scroll
- 📘 **Digital Brochure** — Embedded brochure page gallery
- 👥 **Secretariat Profiles** — Secretary General and team portraits

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Structure | Semantic HTML5 |
| Styling | Custom CSS3 with CSS variables, animations |
| Interactivity | Vanilla JavaScript |
| Fonts | Google Fonts (Cinzel, Bebas Neue, Inter) |
| Hosting | GitHub Pages with custom domain (CNAME) |

## 🚀 Getting Started

```bash
git clone https://github.com/rishitmanktala/colloquium.git
cd colloquium
# Open index.html in a browser
```
